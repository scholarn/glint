# Glint - A Simple and Beautiful Jekyll Theme

Welcome to **Glint**, a Jekyll theme designed with simplicity and beauty in mind. Whether you are building a personal blog, a portfolio, or a business website, Glint offers an elegant and responsive design that will enhance your online presence.

## Features

- **Simplicity**: Clean and minimalistic layout that keeps your content in focus.
- **Responsive Design**: Optimized for all devices, ensuring a seamless experience on mobile, tablet, and desktop.
- **Customizable**: Easily tweak styles and configurations to match your brand or personal aesthetic.
- **Lightweight**: Fast loading times and optimized performance for a better user experience.
- **Open Source**: Freely available for use and modification under the MIT License.

## Getting Started

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/scholarn/glint.git
   ```

2. Navigate to the project directory:
   ```bash
   cd glint
   ```

3. Install the required gems:
   ```bash
   bundle install
   ```

4. Run the Jekyll server:
   ```bash
   bundle exec jekyll serve
   ```

Visit http://localhost:4000 in your browser to see your site in action!

### Usage

To customize Glint for your own site, edit the _config.yml file. Here you can change the title, description, and other settings to fit your needs. You can also modify the layout and styles by editing the relevant files in the /_layouts, /_includes, and assets directories.
